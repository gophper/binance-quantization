## 🎯 核心特性详解

### 1. 智能多指标组合 🎯

```python
# 多指标权重评分示例
买入信号评分：
  - RSI超卖 (权重: 0.25) + 0.25分
  - MACD金叉 (权重: 0.25) + 0.25分
  - 布林带突破 (权重: 0.2) + 0.2分
  - KDJ超卖 (权重: 0.15) + 0.15分
  - 成交量放大 (权重: 0.15) + 0.15分
  ────────────────────────
  总评分 >= 0.7 时触发买入
```

### 2. 分层确认机制 🔍

```
第一层：技术指标信号
  ↓
第二层：成交量确认
  ↓
第三层：大趋势过滤
  ↓
第四层：动态仓位管理
  ↓
触发交易信号
```

### 3. 完整的风险控制 🛡️

```
风险限制：
├─ 单仓位限制: 30%
├─ 单日亏损限制: 2%
├─ 最大回撤限制: 15%
├─ 自动止损: 3%
└─ 自动止盈: 5%

监控指标：
├─ 未实现盈亏
├─ 最大资金回撤
├─ 当日累计亏损
└─ 连续亏损次数
```

### 4. 高保真回测 📊

```
回测精度：
├─ K线级回测 (支持多时间框架)
├─ 手续费模拟 (默认0.02%)
├─ 滑点模拟 (默认0.01%)
├─ 订单执行模拟
└─ 持仓管理模拟

性能指标 (20+)：
├─ 收益指标: 总收益、年化收益
├─ 风险指标: 最大回撤、夏普比率
├─ 交易指标: 胜率、盈亏比
└─ 交易统计: 总数、盈利数、亏损数
```
